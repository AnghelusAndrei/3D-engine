# 3D-engine
c++ 3D engine, made with SDL, supports:
  -basic mesh rendering, 
  -loading .obj files,   
  -basic lighting, 
  -cameras, 
  -mesh rotation, 
  -translation
  -clipping,
  -depth buffer
  
compile with `g++ main.cpp -lSDL2 -lSDL2_image --std=c++17` and run with `./a.out`

you must have SDL installed
